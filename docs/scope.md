# Scope

## In Scope

- Phone-based voice ordering for a single restaurant
- Multi-turn conversation: item selection, modifiers, disambiguation, upselling, confirmation
- Menu grounding with structured validation
- SMS/WhatsApp order confirmation + post-call feedback
- Human handoff with full transcript and partial order context
- POS integration via structured JSON
- Analytics dashboard
- Evaluation framework (task success, system performance, user satisfaction)

## Out of Scope

- Multi-language support (Urdu/English code-switching)
- Drive-through ordering
- Payment processing and loyalty programs
- Caller memory / repeat order recognition
- Multi-restaurant deployment
- White-label configuration portal UI
